WPGens WordPress React Admin Panel is professional looking WordPress administration panel for plugins built with [WP REST API](https://v2.wp-api.org/) and [React](https://reactjs.org/). Its simple to extend and use for your own WordPress plugins. Check out the demo [here.](https://wpgensdemo.com/wp-admin/admin.php?page=gens-raf)
